CSS (Cascading Style Sheets) is a stylesheet language used to define the presentation of a web page. It controls the layout, colors, fonts, and overall look of a website, making it an essential tool for web development. By separating content (HTML) and style (CSS), web developers can create clean, maintainable, and visually appealing websites.

# What is CSS?

CSS stands for Cascading Style Sheets, and it's used to describe how HTML elements are to be displayed on the screen. It allows developers to customize the appearance of a website, making it look polished and professional. CSS can apply to text formatting, colors, spacing, positioning, and much more.

## Basic CSS Syntax

CSS consists of selectors and declaration blocks. Here's an example of the basic syntax:

```css
selector {
  property: value;
}
```
